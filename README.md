# typeORM 으로 게시판 사이트 BE 만들기

# 기능
- 회원가입 및 로그인
- 게시판 전체 보기 페이지네이션
- 게시판 글 등록(사진 포함) (로그인한 사용자만 가능)
- 게시판 글 수정 및 삭제 (로그인한 사용자만 가능)
- 댓글 기능 (로그인한 사용자만 가능)
- 댓글 전체 보기 페이지네이션

# 개발 환경
- IntellJ IDEA (API 개발 개발환경)
- Data Grip (DB 접속 툴)
- Docker (로컬 DB 구동)
